Detailed Description

template<class T>
class llfio_v2_xxx::map_view< T >

Provides a lightweight typed view of a map_handle, a mapped_file_handle or a mapped<T> suitable for feeding to STL algorithms or the Ranges TS.

This is the correct type to use when passing non-owning views of mapped data around between functions. Where you wish the view to be (possibly) owning, you may find the non-lightweight mapped<T> of more use.
