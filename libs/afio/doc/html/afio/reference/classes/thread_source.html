thread_source

Abstract base class for a source of thread workers.

Description

This instantiates a boost::asio::io_service and a latchable boost::asio::io_service::work to keep any threads working. Note that in Boost 1.54, and possibly later versions, boost::asio::io_service on Windows appears to dislike being destructed during static data deinit, hence why this inherits from std::enable_shared_from_this<> in order that it may be reference count deleted before static data deinit occurs.
