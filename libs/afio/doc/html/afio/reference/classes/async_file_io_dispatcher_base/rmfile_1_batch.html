rmfile (batch)

Schedule a batch of asynchronous file deletions after optional preconditions.

Synopsis 
virtual std::vector<async_io_op> async_file_io_dispatcher_base::rmfile(const std::vector< async_path_op_req > & reqs)
Parameters

Type

Concept

Name

Description

const std::vector< async_path_op_req > &

reqs

A batch of async_path_op_req structures.

Returns

A batch of op handles.

Header

#include <boost/afio/afio.hpp>

Complexity

Amortised O(N) to dispatch. Amortised O(N/threadpool) to complete if file deletion is constant time.

Exception Model

Propagates exceptions of any input preconditions with an errored state at the point of dispatch, and throws a std::runtime_error if any inputs have values which could not possibly be correct. Once a batch of input ops has been verified at the point of entry as not errored, you are guaranteed that the batch is atomically scheduled as a whole, unless one of the following conditions occurs: (i) A failure to allocate memory occurs.
