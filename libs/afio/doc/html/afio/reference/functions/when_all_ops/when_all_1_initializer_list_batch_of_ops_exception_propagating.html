when_all (initializer_list batch of ops exception propagating)

Returns a result when all the supplied ops complete. Propagates exception states.

Synopsis 
future<std::vector<std::shared_ptr<async_io_handle> > > when_all(std::initializer_list< async_io_op > _ops)
Parameters

Type

Concept

Name

Description

std::initializer_list< async_io_op >

_ops

A std::initializer_list<> of async_io_op's to wait upon.

Returns

A future vector of shared_ptr's to async_io_handle.

Header

#include <boost/afio/afio.hpp>

Complexity

Amortised O(N) to dispatch. Amortised O(N/threadpool) to complete, but at least one cache line is heavily contended between threads.

Exception Model

Propagating
