when_any (from WG21 N3634 (Improvements to std::future<T> and Related APIs) for range of futures)

Returns a future tuple of results from all the supplied futures.

Description

Makes a future result from the first of the supplied futures to become available

This is an implementation of WG21 N3634's when_any() which uses process_threadpool() to allow you to create a future which only becomes available when the first of the supplied futures become available.
