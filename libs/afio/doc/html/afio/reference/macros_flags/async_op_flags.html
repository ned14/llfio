async_op_flags

Bitwise async_op_flags flags.

Synopsis 
enum async_op_flags {None = 0, DetachedFuture = 1, ImmediateCompletion = 2};
Values

Value

Description

None

No flags set.

DetachedFuture

The specified completion routine may choose to not complete immediately.

ImmediateCompletion

Call chained completion immediately instead of scheduling for later. Make SURE your completion can not block!

#include <boost/afio/afio.hpp>
