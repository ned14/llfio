async_data_op_req< void >

A convenience bundle of precondition, data and where for reading into a void *. Data MUST stay around until the operation completes.

Synopsis 
struct async_data_op_req< void >
      : public detail::async_data_op_req_impl< false >
{
  std::vector< boost::asio::mutable_buffer > buffers;  // A sequence of mutable Boost.ASIO buffers to read into. 
  async_io_op precondition;                            // A precondition containing an open file handle for this operation. 
  off_t where;                                         // The offset from which to read. 
};
