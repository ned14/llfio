Compilation

Boost.AFIO is a headers-only library. Users only need to include the library headers in their programs in order to be able to access definitions and algorithms provided by the Boost.AFIO library. No linking against any binaries is required.

Boost.AFIO is only dependant on headers-only Boost libraries. It does not introduce indirect dependencies on any binary libraries.

In order to be able to use Boost.AFIO, the only thing users need to do is to download and/or install Boost and specify location to include directories, so include directives of this scheme will work: 

#include <boost/...>
